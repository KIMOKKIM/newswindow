# 미리보기 날짜·시간 표시 규칙

## 선택한 표시 형식

- **표시 형식 (렌더링 결과):** `YYYY-MM-DD HH:mm:ss`
- 예: `2026-04-05 22:57:31`

## 근거

- 백엔드 `backend/db/articles.js`의 `nowStr()`가 `toISOString()`을 `T`→공백 치환 후 `slice(0, 19)`로 저장하므로, 저장 값과 **같은 구분자 체계(하이픈 날짜 + 공백 + 콜론 시각)** 로 맞춤.

## 타임존

- 파싱: 첫 공백을 `T`로 바꾼 뒤 `new Date(...)` (기존 날짜-only 미리보기와 동일한 파싱 경로).
- 표시: `Date`의 **로컬** 연·월·일·시·분·초 컴포넌트를 패딩해 조합.

## 빈 값

- 원본이 없으면 표시 문자열은 기존 메타와 동일한 대시 `—` (UI 깨짐 방지).
