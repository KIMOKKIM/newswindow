# ARTICLE_META_VERIFICATION

## 수동 확인 항목

- [ ] `category`가 정확히 `경제-금융`인 기사: 상세 패널 메타에 **금융**만 표시
- [ ] `author_name`이 정확히 `김기목`인 기사: 상세 패널에 **김기목 기자**
- [ ] 발행·수정 일시 포맷 **변경 없음**
- [ ] 다른 카테고리/기자 문자열 **그대로** 표시
