# 좌우 사이드 광고 미표시 — 직접 원인 (1개)

## 확정 원인

**API `sideLeftStack` / `sideRightStack` 및 `sideLeft` / `sideRight`의 이미지 URL(`src`) 필드가 운영 저장 데이터 기준으로 모두 비어 있다.**

- 프론트의 `hidden`·`src` 제거는 **`src`가 빈 경우**에만 수행되는 정상 분기이다.
- 푸터·헤더에는 이미지가 있으나, **좌우 전용 필드에는 값이 들어 있지 않았다** → 플레이스홀더만 노출.

## 조치 (백엔드)

- 공개 `GET /api/ads` 응답에서만, 위 조건(사이드 전부 비어 있음)일 때 **`footer` 항목의 `image`·`href`로 스택을 보강**해 메인에서 이미지가 보이게 함. 저장 `PUT` 경로는 변경하지 않음.
