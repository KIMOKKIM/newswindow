# 메인페이지 카테고리 전체 목록 (canonical)

**원본**: `index.html` 주 메뉴 + `.section-subcats` 순서, 저장용 `value`는 `shared/articleCategories.json`과 동일.

정렬: 상단부터 **정치 → 경제 → 사회 → 문화 → 전국 → 이슈 → (단독) 국제 · 칼럼 · 포토뉴스&영상**.

하위 구조는 **한 단계**(대분류-소분류)이거나, 단독 카테고리는 `value`에 하이픈 없음(`국제`, `칼럼`, `포토뉴스&영상`). `script.js`의 `majorCategory`는 `split('-')[0]`이므로 `포토뉴스&영상`은 통째로 첫 토큰.

| 순서 | 대분류(섹션) | 소분류(메인 노출명) | 저장 value |
|------|-------------|---------------------|------------|
| 1–4 | 정치 | 최신기사, 지방자치, 외교, 북한 | 정치-최신뉴스, 정치-지방자치, 정치-외교, 정치-북한 |
| 5–8 | 경제 | 금융, 부동산, AI/IT, 자동차 | 경제-금융, 경제-부동산, 경제-AI/IT, 경제-자동차 |
| 9–12 | 사회 | 시사, 건강/의료, 교육, 리조트&테마 | 사회-시사, 사회-건강/의료, 사회-교육, 사회-리조트&테마 |
| 13–16 | 문화 | 대중음악, 행사&공연, 갤러리, 문학 | 문화-대중음악, 문화-행사&공연, 문화-갤러리, 문화-문학 |
| 17–23 | 전국 | 서울, 경기, 강원, 충청, 영남, 호남, 제주 | 전국-서울 … 전국-제주 |
| 24–26 | 이슈 | 연예&스포츠, 뷰티, 여행&음식 | 이슈-연예&스포츠, 이슈-뷰티, 이슈-여행&음식 |
| 27 | — | 국제 | 국제 |
| 28 | — | 칼럼 | 칼럼 |
| 29 | — | 포토뉴스&영상 | 포토뉴스&영상 |

**소계**: 소분류(실사용 value) **29개** + 양식용 “선택하세요” 1개.

> 참고: 예전 `nw-office/article-write.html`에만 있던 `뉴스포커스`는 메인 네비/섹션에 없어 canonical에서 제외함.
