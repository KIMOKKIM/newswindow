## 메인·canonical에만 있고 수정 전 admin에 없던 항목 (18개)

- 정치-외교, 정치-북한  
- 경제-부동산, 경제-자동차  
- 사회-건강/의료, 사회-교육, 사회-리조트&테마  
- 문화-행사&공연, 문화-갤러리, 문화-문학  
- 전국-경기, 전국-강원, 전국-충청, 전국-영남, 전국-호남, 전국-제주  
- 이슈-뷰티, 이슈-여행&음식  

## 수정 전 admin에만 있던 항목

- 없음 (부분집합이었음).

## 순서·이름

- 수정 전: 단순 플랫 리스트, 메인의 대분류 순서·그룹과 불일치.  
- 수정 후: `optgroup`으로 메인과 같은 대분류 순서.

## 수정 후

- admin 선택지 **29 value** = 메인 소분류 **29**와 일치(`shared/articleCategories.json`).
